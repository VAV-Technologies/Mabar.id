# Mabar Assets

This folder should contain the following image assets:

## Required Images

### App Icon
- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024px)

### Splash Screen
- `splash-icon.png` - Splash screen icon (200x200px minimum)

### Favicon (Web)
- `favicon.png` - Web favicon (48x48px)

## Design Guidelines

### Brand Colors
- Primary: #FF6B35 (Orange)
- Background Dark: #0A0A0A
- Background Light: #FFFFFF
- Background Cream: #FFFBF5

### Logo Concept
The Mabar logo should incorporate:
- A spinning wheel or roulette element
- Food/dining iconography
- Modern, playful aesthetic

## Generating Assets

You can use tools like:
- Figma (design tool)
- Canva (quick designs)
- Adobe Illustrator (professional)

Or online generators:
- https://icon.kitchen (app icons)
- https://realfavicongenerator.net (favicons)

## Temporary Placeholders

For development, you can use solid color placeholders:
- icon.png: Orange (#FF6B35) square
- splash-icon.png: White text "MABAR" on dark background
- adaptive-icon.png: Same as icon.png
